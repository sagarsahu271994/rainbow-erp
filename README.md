# Rainbow The Learner Zone ERP

Upload rainbow-erp.html to GitHub and rename it to index.html before deploying on Vercel.

Features: Dashboard, Students, Fees Reminders, Admission PDF print, Fees Receipt PDF print, Attendance, Reports, Settings, CSV export, WhatsApp buttons.

Note: Data is saved in Supabase tables: `students`, `admissions`, `fees_receipts`, and `attendance`. Fully automatic WhatsApp reminders need WhatsApp Business API and backend scheduler.

## Default Login

- Username: admin
- Password: Rainbow@123

Change these values inside `index.html` before making the site public. Search for `AUTH_USER` and `AUTH_PASS`.
